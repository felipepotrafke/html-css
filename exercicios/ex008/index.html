Esta frase possui um termo em negrito usando a tag <b> (não semântica)

Esta frase possui um termo em destaque usando a tag <strong> (semântica)


Itálico e Ênfase

Esta é uma frase com um termo em itálico usando a tag <i> (não semântica)

Nesta frase temos um termo em ênfase usando a tag <em> (semântica)

Texto marcado

É possível criar um Texto Marcado em um texto do site com a tag <mark>.
Ou até formatar a cor necessária específica para um texto marcado

Texto Grande e Pequeno

É possível criar um texto com letras grandes com <big> e letras pequenas com <small>.

Texto deletado

É possível criar um texto excluído se quiser que ele não seja considerado ou para algo mais satírico com a tag <dell>.

Texto inserido

Um texto inserido com a tag <ins> indica que ele foi adicionado depois ou para dar um destaque a mais

Também é possível criar um texto sublinhado com outra tag, a <u>

Texto sobrescrito

Serve para inserir coisas para deixar em cima como x25, uma potenciação de alto nível!

Texto subscrito

Serve para inserir coisas em baixo como H2O, uma fórmula química do dia a dia!
